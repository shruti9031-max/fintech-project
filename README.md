Smart Finance Manager is a web-based FinTech application developed using HTML, CSS, and JavaScript. The main objective of this project is to help users manage their daily finances efficiently by providing features like budget tracking, micro-investing simulation, and a simple payment gateway interface.

This application allows users to add their income and expenses, automatically calculates the remaining balance, and helps them understand their spending habits. It also includes a micro-investment feature where users can invest small amounts and observe simulated growth, encouraging financial awareness and saving habits.

Additionally, a basic payment gateway interface is implemented to demonstrate how online transactions work in a user-friendly environment. Although it does not perform real transactions, it gives a clear understanding of the payment process.

The project focuses on improving financial discipline, promoting smart money management, and providing a beginner-friendly FinTech solution. It also demonstrates the practical implementation of front-end web development concepts such as DOM manipulation, event handling, and dynamic data updates.
